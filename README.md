# Real-time Chat Server
### Goal & Description of this project 
To Create a real-time chat server using Flask and Websockets, 
so that people can send messages and receive updates instantly. 
& To :
- [x] Learn more about Websockets
- [ ] Apply Concepts Learned from Code Complete
    - [ ] Routines & Cleaner Code
- [ ] Get better with React & GUI 
- [ ] More practice with SQLite

### Bonus Features
- [ ] Add Usernames so messages show who sent them 
- [ ] Store the last 10 messages so new users can see some past chats
- [ ] Deploy it to a free hosting service, maybe Vercel
- [ ] Allow people to draw + Draw on images and send it to another person. 

# Running the Project
### Installing & creating a new venv to project
You should specifically be in the folder for this

Enter the following commands.
`Python3 -m venv .venv`
`Source .venv/bin/activate`
`pip3 install -r requirements.txt`

Installing React & running the frontend
`cd chatterbox`
`npm install`
`npm run dev`
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

# Updating the Project
### Updating Requirements.txt
`pip3 freeze > requirements.txt`
OR
`pip3 install pipreqs`
`pipreqs /path/to/project`
